Theory

The selection sort algorithm sorts an array by repeatedly finding the minimum element (considering ascending order) from unsorted part and putting it at the beginning. The algorithm maintains two subarrays in a given array.

Pseudo-code
selectionsort(A){where A is the array to be sorted}

  1. For i = 1 to A.length - 1
    Min = i
  2. For j = i+1 to A.length
    If A[min] > A[j]
    Min = j
  3. Swap A[min] and A[i]

Time-complexity: O(n^2)
Proof of Time Complexity
In the worst case, for every iteration of the outer loop, the inner loop would have to search till the end of the array to find the minimum element for that iteration.
