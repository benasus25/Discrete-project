Theory

Binary Search is a faster way of searching elements in a sorted array.
Binary Search Algorithm can be implemented in two ways - the iterative and the recursive method. The recursive method follows the divide and conquer approach.
Both the methods have been discussed below.

Pseudo-code for iterative method
binarySearch(arr, x, low, high) {where arr is the array to be searched, x is the element to be searched, low is the index of the first element and high is the index of the last element.}

  1. Run the loop until the pointers low and high meet each other.
  2. mid = (low + high)/2
  3. if (x == arr[mid]) return mid
  4. else if (x > A[mid]) low = mid + 1
  5. else high = mid - 1

Pseudo-code for recursive method
binarySearch(arr, x, low, high) {where arr is the array to be searched, x is the element to be searched, low is the index of the first element and high is the index of the last element.}

  1. if low > high return False
  2. else mid = (low + high) / 2
  3. if x == arr[mid] return mid
  4. else if x < data[mid] return binarySearch(arr, x, mid + 1, high)
  5. else return binarySearch(arr, x, low, mid - 1)

The time complexity of Binary Search is O(logn).
The space complexity of Binary Search is O(n).
