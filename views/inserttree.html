Theory

This is used to insert a specific node in binary search tree.

Pseudo-Code
Tree-Insert(T, z){where T is the root node and z is the element to be inserted.}

  1. y = NULL, x = T.root
  2. While x != NULL, y =x
    If z.key < x.key, x =x.left
    else x = x.right
  3. z.p = y
  4. If y == NULL, T.root = z
  5. elseif z.key < y.key, y.left = z
  6. else y.right = z

Time-Complexity: O(h), where h is the height of the tree
Proof of Time Complexity -
Just like, Tree-Search, Tree-Insert begins at the root of the tree and the pointer x traces a simple path downward looking for a NULL to replace it with z.
