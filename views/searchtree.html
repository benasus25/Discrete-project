Theory

This is used to search a particular value in a binary search tree

Pseudo-Code
Tree-Search(x, k){where x is the root node and k is the element to be searched}

  1. if x == NULL or x.key == k, return x
  2. if k < x.key, return Tree-Search(x.left, k)
  3. if k>x.key, return Tree-Search(x.right, k)

Time complexity: T(n) = O(h), where h is the height of tree
Proof of Time Complexity
The procedure begins its search at the root and traces a simple path downward in the tree and thus the distance travelled/nodes traversed are only dependant on the height of the tree.
